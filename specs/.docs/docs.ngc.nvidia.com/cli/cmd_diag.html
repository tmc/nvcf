diag¶

The diagnostic commands print configuration and environment information and provide suggestions on identifiable misconfigurations or common issues. No private credentials are included in the gathered diagnostic data.

usage: ngc diag [--debug] [--format_type <fmt>] [-h]
                {all,client,install,server} ...

Named Arguments¶

--debug

Enable debug mode.

--format_type

Possible choices: ascii, csv, json

Specify the output format type. Supported formats are: ['ascii', 'csv', 'json']. Only commands that produce tabular data support csv format. Default: ascii

diag¶

{all,client,install,server}

Possible choices: all, client, install, server, user

Sub-commands¶

all¶

Produce the maximum amount of diagnostic output (client, install, server, user).

ngc diag all [--debug] [--format_type <fmt>] [-h]
